> ### _Next.js_ is a React framework for building full-Stack Web Applications. You use React Components to build user interfaces, and Next.js for additional features and optimizations. 
- It aids in creating web apps with server-rendering and statically-generated web applications, offering features to improve development and performance.
- You still write React codes and use React features. NextJS just enhances your React apps and adds more features.

> [!IMPORTANT]
>> #### Key Features & Benefits :
>> _` Server-side Rendering `_: <br/> Automatic page re-rendering - Great for SEO and initial load <br/> Blending Client-side and Server-side: Fetch data on the server and render finished pages <br/> <br/>
>> _` File-based Routing `_: <br/> Define pages and routes with files and folders instead of code <br/> Less code, less work, highly understandable <br/> <br/>
>> _` Fullstack Capabilities `_: <br/> Easily add Backend (Server-side) code to your Next/React apps <br/> Storing data, getting data, authentication etc. cab be added to your React projects
